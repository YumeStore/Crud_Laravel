## Instalação:

- Crie um banco de dados mysql como o nome "laravel"
- Copie copie o arquivo .env-example e crie um novo arquivo chamado .env 
- no terminal digite: composer install
- npm install
- php artisan migrate
- php artisan serve

 O site deve estar rodando em "localhost:8000"
